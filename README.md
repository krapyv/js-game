# 2D Game 🎮

A canvas-based side-scroller action game built with vanilla JS.

**Live Demo:** [krapyv.github.io/js-game](https://krapyv.github.io/js-game)

## Controls

| Key | Action |
|-----|--------|
| W | Jump |
| A/D | Move left/right |
| Enter | Run |
| Q/R/Z | Special attacks |
| S | Stop |

## Features

- 4 enemy types (Digger, Hand, Raven, Zombie)
- Player states (walk, run, jump, hit, dead)
- Parallax scrolling background
- Score system + timer
- Collision detection with explosions

## Run Locally

```bash
git clone https://github.com/krapyv/js-game.git
cd js-game
open index.html
```
